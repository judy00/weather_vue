### 簡介
- - -
此作品為 Open Weather API + Vue Cli 前端練習，以及使用到 Highcharts 製作圖表
* 可以在搜尋框輸入各國城市名稱，或是直接點擊範例城市名稱做搜尋
* 可以切換華氏 (ºF) 與攝氏 (ºC) 單位
* 顯示當日每三小時天氣情況與未來一周預報
